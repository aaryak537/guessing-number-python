# Guess the Number Game (Python)

## Description
Guess the Number is a simple terminal game written in Python.  
The program generates a random number, and the player must guess it within a limited number of attempts.  
This project is designed as to practice Python fundamentals such as loops, conditionals, and modules.

## Features
- Random number generation
- Limited guessing attempts
- Clear user prompts
- Feedback for each guess (Too High / Too Low)
- Simple and readable code

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the program
